Web Development

Web development ranges from developing a simple single static page website of plain text to ultra-complex web-based works like internet applications, electronic businesses or social network services. Web development is the coding or programming part of website functionality that mainly deals with the non-design aspect of building websites. There are two divisions of web development- Front-end development (client-side development) and back-end development (server-side development). Front-end development refers to constructing what a user sees when they visit a website using codes such as HTML, CSS, and JavaScript. Back-end development controls behind the scenes of a web application using a database to generate the front-end. Back-end scripts are written in many codes and frameworks, such as PHP, Java, Node.js, Python, etc.
